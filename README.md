# Universal Questionnaire Engine (UQE)


## Overview

Developed a configurable rule-based workflow engine to support **dynamic questionnaire systems**, real-time decision handling, and score-driven automation workflows.
The platform enabled adaptive questionnaire flows with configurable rules, red-flag detection, and conditional branching without requiring code-level modifications.



## Features

* Dynamic questionnaire workflow engine
* Priority-based rule processing
* Red-flag detection and alert handling
* Tier-based adaptive questionnaire flows
* Conditional branching and workflow transitions
* Score-driven automation logic
* Multi-language configuration support
* Dynamic report generation
* Config-driven architecture with zero-code updates



## Tech Stack

* Python
* JSON Configuration
* REST APIs



## Workflow

1. Load questionnaire configuration from JSON
2. Process user responses dynamically
3. Apply priority-based rules and scoring logic
4. Trigger conditional workflow transitions
5. Detect red-flag conditions in real time
6. Generate adaptive questionnaire paths
7. Produce dynamic reports and outputs
